# unianalytic

## Description
Ce dataset contient des données fictives d'étudiants pour tester l'application UniAnalytics.

## Fichiers

- etudiants.csv : informations sur les étudiants
- resultats.csv : notes des étudiants

## Colonnes principales

### etudiants.csv
- id_etudiant
- matricule
- nom
- prenom
- filiere
- niveau
- age

### resultats.csv
- id_etudiant
- nom_matiere
- note_examen

## Utilisation
Importer ces fichiers dans votre application Streamlit ou base SQLite pour tester les analyses.
